What this project is

This Expo app uses your enhanced @nozbe/watermelondb package and exposes a simple Supabase-like reactive API for local SQLite data. Components auto-update with useReactiveQuery without manual subscription wiring.

Routing note: this site is now rebuilt with static HTML and includes compatibility redirects for /docs/* URLs that were previously returning 404.

Quick start

npm install
npm run setup:db:vendor
npx expo run:android

# checks
npx tsc --noEmit
CI=1 npx expo export --clear --platform android

To consume the published enhanced package instead of vendored tarball: npm run setup:db:npm
