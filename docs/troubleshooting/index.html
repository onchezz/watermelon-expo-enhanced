Page Not Found on GitHub Pages

This site uses GitHub Pages source main:/docs. Root URL is: https://onchezz.github.io/watermelon-expo-enhanced/

Older links that included /docs are now handled by compatibility redirects included in this rebuild.

Syntax error from Flow decorators (`@lazy`)

Cause: imports from internal package source path, for example @nozbe/watermelondb/src/....

Fix: use package entrypoints only.

@nozbe/watermelondb
@nozbe/watermelondb/adapters/sqlite
@nozbe/watermelondb/reactive/react

TypeScript: import type only in TS files

Fix by using .ts/.tsx files for type-only imports, or use normal import.

TS5053: emitDeclarationOnly with noEmit

These options cannot be enabled together in the same TypeScript config run.
